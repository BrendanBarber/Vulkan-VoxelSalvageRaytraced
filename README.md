# Voxel Raytracing (with some Raymarching)

Renders primitive shapes in a voxel-style and customizable voxel size. Currently in the process of rewriting it for the Vulkan Raytracing Pipeline. This is a brute-forced, messy project that I used to learn raytracing math.
